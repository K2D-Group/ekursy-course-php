# Tworzenie katalogów
Do tworzenia katalogów słóży funkcja `mkdir()`, która przyjmuje jako parametr nazwe katalogu.

##Przykład:
Program utworzy katalog o podanej nazwie.

```php
$katalog = "katalog";

mkdir($katalog);
```
