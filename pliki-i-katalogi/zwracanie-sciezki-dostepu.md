# 8. Zwracanie śceżki dostępu

Funkcja `dirname(/* Nazwa */)` zwraca ścieżkę pliku.

##Przykład
Program wypiesze ścieżkę do istniejącego pliku.

```php
$nazwa = "index.php";
$sciezka = dirname($nazwa);
echo $sciezka;
```
